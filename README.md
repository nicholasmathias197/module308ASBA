
# Module 308 ASBA

This is a simple Dog Breed Selector app. It lets users browse dog breeds fetched from an external API and add their favorite breeds to a favorites list.

**How it works:**
- Open the app in your browser (index.html)
- Browse a list of dog breeds (fetched from the Dog API)
- Add breeds to your favorites
- View and manage your favorite breeds
